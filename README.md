# SecureAPI-Express
C' est une API développée avec Express .js qui intègre des mesures de sécurité avancées : ✔Protection des en-têtes HTTP avec Helmet ✔Limitation du nombre de requêtes pour éviter les attaques DDoS (express-rate-limit)  ✔Gestion avancée des logs avec Winston 📝 ✔Masquage des mots de passe dans les logs pour plus de confidentialité.
